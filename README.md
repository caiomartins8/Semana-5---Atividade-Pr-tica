## Responsividade (CSS Puro)

O projeto foi desenvolvido seguindo a estratégia de adaptação de layout:
- **Desktop:** Menu horizontal e cards em 3 colunas.
- **Mobile:** Menu vertical e cards em coluna única (100% de largura).
